FIPS 199 categorization tells us how sensitive this system is. Think about what would happen if something went wrong — if data was stolen, altered, or the system went offline. Your answers here determine how many security controls are required. When in doubt, select the level that matches the worst realistic scenario for your agency.

Low — limited harm if data is disclosed (e.g., public info).
Moderate — serious harm if data is improperly disclosed (e.g., employee PII).
High — severe or catastrophic harm (e.g., classified data, financial records, health info).

Low — minor harm if data is altered (e.g., internal notes).
Moderate — significant harm if data is corrupted or tampered with (e.g., employee records).
High — catastrophic harm if data is altered (e.g., financial ledgers, safety-critical systems).

Low — it's OK if the system is down for a while (e.g., internal reporting tool).
Moderate — outages cause significant disruption (e.g., email, internal HR portal).
High — any downtime is unacceptable (e.g., 911 dispatch, emergency response systems).

{# ════════════════════════════════════════════════════════════════════════ SECTION 6 — Authorization (AO / Admin only) ════════════════════════════════════════════════════════════════════════ #} {% if user_role in ('ao', 'admin') %}
Authorization

An Authorization to Operate (ATO) is the official management decision to allow a system to operate. It is granted by an Authorizing Official (AO) — typically a senior agency executive — after reviewing the system's security posture. ATOs are typically valid for 3 years, after which they must be renewed. If your system is new and hasn't been through this process yet, select "Not Authorized."

Not Authorized — the system has not yet gone through the formal authorization process.
In Progress — authorization package is being prepared or is under review.
Authorized — the system has a valid, active ATO signed by an Authorizing Official.
Expired — the ATO existed but has lapsed; the system needs re-authorization.

The date the Authorizing Official signed the ATO decision letter. Leave blank if not yet authorized.

When the current ATO expires. Typically 3 years from the authorization date. BLACKSITE will alert you when expiry is approaching.
