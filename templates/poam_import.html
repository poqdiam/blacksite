{# Tips #}
Import Tips
  • Save your spreadsheet as CSV (UTF-8). Excel BOM is handled automatically.
  • Leave system_name blank to link all rows to the selected system below, or fill it per row to assign to different systems.
  • Use Dry Run to preview results without saving — check for errors before committing.
  • Invalid severity values default to Low. Invalid dates are cleared.
  • There is no duplicate detection — re-importing the same file will create duplicates. Use dry run to verify first.
