Deployment Profiles

Pre-configured compliance program bundles. Activating a profile enables its framework set in the org framework catalog and sets the recommended workflow terminology. Multiple profiles can be active simultaneously for multi-framework deployments.

How it works: Activating a profile enables its frameworks in Framework Management so they appear in the Control Catalog and system selection menus. It does not disable other active frameworks — you can combine profiles freely. Deactivating only marks the profile inactive; it does not disable frameworks.
